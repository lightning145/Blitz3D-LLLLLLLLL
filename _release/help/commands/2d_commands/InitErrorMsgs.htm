InitErrorMsgs size, [hasMacro]

Parameters

size = size of message
hasMacro = enable macros

Description

Initialize custom dialog of memory access violation.
Size is lines amount of the dialog, except this, you can also use Chr(10) to line feed.
If macro is enabled, then replace some string with specify values.
Available Marcos:
_CaughtError_: Latest caught exception
_AvailPhys_: Available physical memory
_AvailVirtual_: Available virtual memory
