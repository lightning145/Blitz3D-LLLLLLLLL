GetDomainTXT domain$

Parameters

domain = TXT answer source domain name

Description

Get TXT answer from domain, use for feature such as check update.

If the last character of string isn't semicolon(;), then add automaticly.

See Also: ParseDomainTXT

Example

txt$ = GetDomainTXT("version.scpcbgame.cn") ; get txt answer
Print ParseDomainTXT(txt, "version") ; parse and get value of "version"
Print ParseDomainTXT(txt, "date") ; parse and get value of "date"
WaitKey
End
