SetErrorMsg pos, message

Parameters

pos = pos of message
message = self-evident

Description

Set a messagebox of the specify line of custom dialog of memory access violation.
If macro is enabled, then replace some string with specify values.
Available Marcos:
_CaughtError_: Latest caught exception
_AvailPhys_: Available physical memory
_AvailVirtual_: Available virtual memory
