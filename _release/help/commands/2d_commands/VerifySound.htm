VerifySound (sound_variable)

Parameters

sound_variable = variable previously assigned with a LoadSound command.

Description

This command verifies the integrity and validity of the sound variable. Returns True if the sound variable is valid and can be used for playback, or False if the sound variable is released or invalid.

Example

; Assign a global variable for the sound
Global sound
; Load the sound file into memory
sound = LoadSound("sounds/die.wav")
; If the sound is valid, play the sound, otherwise tell the player that the sound file is invalid.
If VerifySound(sound)
PlaySound(sound)
Else
Print("Oops, the sound file is invalid.")
EndIf
